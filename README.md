# Living Logic — Student 5 Module

This module implements the governance, deployment, logging, and rollback layer
for the Living Logic cyber-resilient ICS framework.

The purpose of this module is to:
- record anomaly and adaptation events,
- maintain deployment traceability,
- support operator approval workflows,
- provide rollback and recovery mechanisms,
- and create an auditable history of system changes.

Technologies used:
- Python
- SQLite
- Git/GitHub
- Python logging library
